项目群总监
岗位职责
  1. 负责完成项目群组团队管理及人才培养；
  2. 负责确定项目方案核心思路，管控项目全过程，对项目结果负责；
  3. 统筹完成客户沟通、方案汇报、成果质量把控及确认；
  4. 配合完成项目经验和专业技能培训分享、知识体系搭建。
任职资格
  1. 本科及以上学历，专业不限。旅游管理、城市规划、景观设计、历史文化等相关专业优先；
  2. 六年以上旅游地产/旅游策划/旅游规划等相关行业丰富的专业知识和项目工作经验；
  3. 丰富的项目经验，在旅游策划/旅游地产方面有较强的理论与实战经验，熟悉项目制公司运作模式，具备独立运作项目的工作经验，保证项目顺利完成，有多项目独立统筹管理经验者优先；
  4. 具备优秀的计划、组织、沟通、领导能力以及质量管理与风险管理能力；
  5. 具备一定的团队管理、商务谈判及客户维护能力；
  6. 具备良好的职业道德、团队合作精神、能够承受一定压力，致力于旅游咨询行业长期发展。

商务总监
岗位职责
  1. 独立负责开展商务工作，签订项目合同，完成全年商务指标；
  2. 开拓并维护客户资源，与合作方保持良好联系，注意维护公司品牌形象；
  3. 熟悉行业市场，收集并整理行业、竞争对手等相关信息；
  4. 领导安排的其他工作；
任职资格
  1. 旅游管理、旅游规划设计、市场营销或相关专业本科以上学历；
  2. 六年以上市场拓展经验，有旅游、景观、规划、建筑等行业经验者优先；
  3. 出色的表达沟通、人际交往能力，能与政府、开发商等建立良好合作关系；
  4. 良好的业务拓展能力，出色的商务谈判技巧；
  5. 较好的责任感及团队精神，在高压下保持积极的工作态度；
  6. 致力于旅游行业，能接受频繁出差。

项目经理
岗位职责
  1. 负责确定项目方案核心思路，把控整体项目团队工作安排及进度，对项目成果负责；
  2. 管控项目的成本及预算，有风险管控意识，预判项目风险并提出建议对策；
  3. 有能力完成客户沟通、汇报、成果提交与确认；
  4. 参与技术沉淀及创新，能提升项目成员的技术及业务能力。
任职资格
  1. 专业不限，本科及以上学历；
  2. 四年以上区域旅游规划、旅游策划、房地产、城市规划、景观等相关行业工作经验；
  3. 丰富的项目经验，熟悉项目制度，能够负责旅游规划项目，保证项目顺利完成；
  4. 具备优秀的计划、组织、沟通、领导能力以及质量管理与风险管理能力；
  5. 具备一定的团队管理、商务谈判及客户维护能力；
  6. 具备良好的职业道德、团队合作精神，能够承受一定压力，致力于旅游咨询行业长期发展。
